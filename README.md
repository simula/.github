# Simula

Welcome to the GitHub organization for **Simula** — a Norwegian research organisation working to solve fundamental problems in information and communication technology (ICT) that benefit society.

Simula conducts high-quality research, contributes to education, and supports innovation through open science, software, datasets, collaboration, and research-based development.

## What we work on

Our work spans core areas of ICT, including:

* Artificial intelligence
* Scientific computing
* Communication systems
* Software engineering
* Cyber security
* Data, infrastructure, and reproducible research

Across these areas, Simula researchers develop methods, tools, experiments, and software that support both fundamental research and real-world applications.

## Open research and software

This GitHub organization hosts open-source software, research prototypes, datasets, infrastructure, and supporting material from projects across Simula.

You may find repositories related to:

* Scientific software and numerical methods
* Machine learning and AI research
* Networked systems and communication technology
* Security, privacy, and resilience
* Reproducible research artifacts
* Open datasets and tools for research communities
* Teaching, training, and research infrastructure

Repository maturity varies: some projects are actively maintained production-quality tools, while others are research artifacts shared to support transparency, reproducibility, and collaboration.

## Using our repositories

Each repository should include project-specific information such as installation instructions, usage examples, license details, citation guidance, and contribution notes.

When using software or data from Simula repositories, please check the individual repository for:

* License terms
* Citation requirements
* Maintenance status
* Issue reporting guidelines
* Contact information

## Collaboration

Simula collaborates with universities, research institutes, public-sector organisations, industry partners, and open-source communities in Norway and internationally.

We welcome responsible use, feedback, issues, pull requests, and research collaboration where appropriate. For project-specific questions, please open an issue in the relevant repository or contact the maintainers listed there.
